# Mailchimp Analyser

Fullstendig Mailchimp-analyse med Supabase backend.

## Edge Functions

- **mailchimp-webhook** — Mottar webhook-events fra Mailchimp
- **mailchimp-sync** — Synkroniserer kampanjer og lister fra API
- **mailchimp-stats** — Returnerer analysert statistikk
